A scalar variable (or scalar field) is a [variable](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/variable.md) which holds one (and only one) *[string](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/string.md) [value](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/value.md)* at all times.
