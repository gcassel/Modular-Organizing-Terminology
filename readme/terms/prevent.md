# Definition

_To prevent_ is _to stop a_ [_specific_](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/specific.md) __ [_action_](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/act.md) _or_ [_activity_](https://github.com/gcassel/Modular-Organization-Terminology/blob/master/terms/activity.md) _from happening._
