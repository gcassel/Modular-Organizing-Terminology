# template

**Definition**

_A template_ is _a_ [_generic_](generic.md) __ [_model_](model.md) which _can be_ [_customized_](customize.md) _to_ [_facilitate_](facilitate.md) __ [_efficient_](efficient.md) __ [_production_](produce.md) _of a_ [_specific_](specific.md) __ [_type_](type.md) _of_ [_resource_](resource.md) in a specific [context](context.md).

**Notes**

Templates can be [used](use.md) to produce [physical](physical.md) resources _or_ any type of [media](media.md).

**Related Terms**

Templates are often [frameworks](framework.md). However, templates don't [necessarily](require.md) [_include_](include.md) __ [_frames_](frame.md), so they're not always frameworks.

Frameworks aren't necessarily templates, but they often include templates and can _be_ templates. For example, a _web framework_ is a [_developmental_](develop.md) _framework_ which often includes default or even "boilerplate" code [modules](module.md), which are customizable templates.
